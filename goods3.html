Белый войлочный кошачий дом с ушами

Описание товара

Невероятно эстетичный белый войлочный кошачий дом с ушами

Характеристики товара

  • Приблизительные размеры: 45 * 45 * 55 см
  • Войлок из натуральной шерсти
  • Натуральные красители
  • Высокие теплоизоляционные свойства
  • Фронтальное входное отверстие

Особенности товара

  • Для стирки дома выбрайте режим "Шерсть"
  • Подвешивайте при сушке для придания правильной формы
  • Дом не предназначен для мышей
  • Не ставить вблизи огня
  • Держать в сухом месте

Дом сделан из органической шерсти.
Для валяния это дома использовались только вода, мыло и натуральная шерсть.
Такой дом выберет требовательная кошка с высокоразвитым чувством прекрасного.
